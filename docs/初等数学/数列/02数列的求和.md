# 数列的求和

本节讲述数列的基本求和方法.

## 常数、等差、等比数列的求和

这是三个基本公式。

* 若$a_n = a$($a$为常数)，则$S_n = na$。
* 若$a_n$为等差数列，也就是$a_n = a_1 + d(n - 1)$，则：

$$
\begin{align}
S_n 
=& \frac{(a_1+a_n)n}{2} \\
=& a_1 n + \frac{dn(n-1)}{2} \\
=& \left( a_1 - \frac{d}{2} \right) n + \frac{d}{2} n^2.
\end{align}
$$

* 若$a_n$为等比数列，也就是$a_n = a_1 q^{n-1}$，则：

$$
S_n = \frac{a_1(1-q^n)}{1-q}
$$
