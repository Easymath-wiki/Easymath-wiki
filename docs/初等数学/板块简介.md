# 板块简介

**初等数学**板块主要介绍各种初高中阶段已经涉及的知识，或者初高中虽然未涉及，但是大学课本默认中学生已经学会的知识。

其中，某些知识在大学数学中有所涉及，甚至经常使用。如果读者对其中某些知识不太熟悉，建议多看一看、自己总结方法。大学老师很多不会再重点强调初等的知识，所以需要大家认真理解哦！

## 大学常用的各类初等内容

以下列出一些常用的知识点，供读者参考。

### 通用内容

* 集合（包括高中很少涉及的差集）
* 映射（高中课本已经删了）
* 命题逻辑（与或非，逆否命题等价性）
* 数学归纳法
* 数列（递推关系求通项，求和）

### 数学分析/微积分

* 三角函数（包括中学很少涉及的余切、正割、余割）
* 反三角函数（包括简单性质）
* 极坐标（与直角坐标的互化）
* 参数方程

### 概率论

* 排列组合知识
