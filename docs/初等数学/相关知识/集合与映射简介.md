# 集合与映射

## 1.1 集合

#### 集合运算

1. 交换律		$A∪B = B∪A$		$A∩B = B∩A$

2. 结合律 		$A∪(B∪C) = (A∪B)∪C$		$A∩(B∩C) = (A∩B)∩C$

3. 分配律		$A∩(B∪C)= (A∩B)∪(A∩C)$ 	$A∪(B∩C)= (A∪B)∩(A∪C)$

4. 对偶律		$(A∪B)^c$ = $A^C$∩$B^C$		$(A∩B)^c$ = $A^C$∪$B^C$

#### 有限集与无限集

若集合S由n个元素组成，n是确定的非负整数，则称S为有限集，反之称为无限集。
