# 数列极限的性质

数列极限有符号表述法, 以后我们会常常用到, 所以熟读并背诵. ~~这真的需要背嘛~~.

$$
\lim_{n \rightarrow \infty}x_n = a \Leftrightarrow \forall \epsilon > 0, \space \exists N, \space \forall n > N: |x_n - a| < \epsilon.
$$

## 定理 2.2.1 (极限的唯一性)

收敛数列必惟一. 书上的证明很详细了.

## 定理 2.2.2

收敛数列必有界. 用极限的定义按照书上证明即可.
注意 **逆命题不成立** , 但是后面会知道对有界数列加以限制就可收敛.
